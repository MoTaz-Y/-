# دليل تشغيل موقع التوأمة الرقمية

## 📦 محتويات الملف المضغوط

بعد فك الضغط عن ملف `digital-twin-vodafone-final.tar.gz` ستجد المجلد:

```
artifacts/digital-twin-vodafone/
├── src/                  ← كود الموقع الكامل
│   ├── pages/            ← صفحات الموقع
│   ├── components/       ← المكونات (Navbar, Footer, الأقسام)
│   └── context/          ← نظام تسجيل الدخول
├── public/               ← الصور والأصول
├── index.html
├── package.json
└── vite.config.ts
```

---

## 🚀 طريقة تشغيل الموقع محلياً

### المتطلبات الأساسية

قبل البدء تأكد من تثبيت:

- **Node.js** (نسخة 18 أو أحدث) → [تحميل من nodejs.org](https://nodejs.org)
- **pnpm** (مدير الحزم) → يُثبَّت بالأمر:

```bash
npm install -g pnpm
```

---

### خطوات التشغيل

**الخطوة 1 – فك ضغط الملف**

```bash
tar -xzf digital-twin-vodafone-final.tar.gz
```

أو استخدم برنامج WinRAR أو 7-Zip على ويندوز.

---

**الخطوة 2 – الدخول إلى مجلد الموقع**

```bash
cd artifacts/digital-twin-vodafone
```

---

**الخطوة 3 – تثبيت المكتبات**

```bash
pnpm install
```

> إذا لم يكن pnpm متاحاً يمكن استخدام npm بدلاً منه:
> ```bash
> npm install
> ```

---

**الخطوة 4 – تشغيل الموقع**

```bash
pnpm dev
```

> أو بـ npm:
> ```bash
> npm run dev
> ```

---

**الخطوة 5 – فتح الموقع في المتصفح**

بعد تشغيل الأمر السابق ستظهر رسالة مثل:

```
VITE ready in 300ms
➜  Local: http://localhost:5173/
```

افتح المتصفح واذهب إلى: **http://localhost:5173**

## 👤 كيفية استخدام نظام الحسابات

### إنشاء حساب جديد

1. اضغط زر **"إنشاء حساب"** في شريط التنقل العلوي
2. أدخل:
   - **الاسم الكامل** (مثال: محمد أحمد)
   - **البريد الإلكتروني** (مثال: mohamed@example.com)
   - **كلمة المرور** (6 أحرف على الأقل)
   - **تأكيد كلمة المرور** (نفس كلمة المرور)
3. اضغط **"إنشاء الحساب"**
4. سيتم تسجيل دخولك تلقائياً والانتقال للصفحة الرئيسية

### تسجيل الدخول

1. اضغط زر **"تسجيل الدخول"** في شريط التنقل
2. أدخل البريد الإلكتروني وكلمة المرور المسجلين مسبقاً
3. اضغط **"تسجيل الدخول"**

### تسجيل الخروج

- بعد تسجيل الدخول يظهر اسمك في شريط التنقل
- اضغط زر **"تسجيل الخروج"** لإنهاء الجلسة

> **ملاحظة:** بيانات الحسابات تُحفظ في المتصفح (LocalStorage) ولا تحتاج إلى قاعدة بيانات أو إنترنت.

## ⚙️ بناء نسخة الإنتاج (اختياري)

لبناء نسخة جاهزة للرفع على استضافة خارجية:

```bash
pnpm build
```

ستجد النتيجة في مجلد `dist/` جاهزة للرفع على أي استضافة ثابتة (مثل Netlify أو Vercel أو GitHub Pages).

## 🛠️ المشكلات الشائعة

| المشكلة | الحل |
|---------|------|
| `pnpm: command not found` | نفّذ `npm install -g pnpm` أولاً |
| المنفذ 5173 مشغول | أضف `--port 3000` لتغيير المنفذ |
| الصفحة لا تفتح | تأكد أن الأمر `pnpm dev` يعمل بدون أخطاء |
| نسيت كلمة المرور | افتح المتصفح → F12 → Application → LocalStorage → احذف `dt_users` |
